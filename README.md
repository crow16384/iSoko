# Sokoban

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](LICENSE)

A Sokoban puzzle game built with Swift and Xcode.

## Features

- Classic Sokoban gameplay with box pushing and goal tiles
- Multiple levels with difficulty tiers
- Pathfinding and push detection logic in game state

## Getting Started

1. Open `Sokoban.xcodeproj` in Xcode.
2. Select the `Sokoban` scheme and an iOS Simulator target.
3. Build and run.

## Project Structure

- `Sokoban/` - app source code
- `Sokoban/Managers/` - app and game management logic
- `Sokoban/Models/` - core game models and state
- `Sokoban/Scenes/` - scene-specific logic
- `Sokoban/Views/` - SwiftUI/UI view layer
